🇯🇵 2025 跨世代東京與近郊五日深度旅遊規劃
本專案旨在為 60-70 歲長輩與 15-50 歲中青年族群提供一套平衡「慢節奏舒適」與「數位打卡熱點」的東京旅遊藍圖。

🗺️ 行程重點摘要
🗓️ 7/10 - 7/11 河口湖組合方案
根據當日天氣與體力，可選三種景點組合方案：

藝文型：音樂盒之森美術館（歐式莊園與自動樂器表演）。

全景型：富士山全景纜車（3分鐘直達山頂）與大石公園（7月薰衣草季）。

遺產型：忍野八海（富士山湧泉名勝）與新倉山淺間公園。

🗓️ 7/12 鐮倉與御殿場購物
文化：搭乘江之電前往長谷寺觀賞大佛。

零售：下午前往御殿場 Premium Outlets，結合富士山美景與高端精品購物。

🗓️ 7/13 迪士尼樂園分流攻略 (10:00-20:00)
長輩專區：豪華馬克吐溫號蒸汽船（12分鐘悠閒航行）、米奇交響樂 3D 劇場 。

青年專區：美女與野獸城堡奇緣（建議購買 DPA）、飛濺山（16公尺俯衝）。

🗓️ 7/14 東京市區與返台
景點：淺草寺雷門參拜（無障礙設施完善）與銀座步行者天國散策 。

🍴 嚴選美食清單
淺草今半別館：百年壽喜燒，適合長輩的精緻晚餐 。

五代目花山烏龍麵：5公分寬麵體，IG/TikTok 打卡熱點 。

不動烏龍麵 (東戀路店)：由保坂猛設計的雲朵建築，視覺與味覺雙重體驗。
